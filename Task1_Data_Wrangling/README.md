# Task 1 - Data Immersion & Wrangling

## Objective
The objective of this task is to acquire, understand, clean, and prepare the dataset for further analysis.

## Dataset Overview
The dataset contains sales transaction information including customer details, product information, quantity purchased, and revenue generated.

## Data Quality Assessment
The following issues were identified:
- Missing values
- Duplicate Order IDs
- Date formatting issues
- Potential outliers

## Data Cleaning & Transformation
- Replaced missing Age values using median
- Replaced missing City values using mode
- Converted Order_Date to datetime format
- Created Sales_Per_Unit feature

## Deliverables
- Data Dictionary
- Cleaning Script
- Cleaned Dataset
